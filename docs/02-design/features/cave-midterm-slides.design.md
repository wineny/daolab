**시각 구성**
- 중앙 대제목: **CAVE**
- 부제: "We go where the map ends"
- 하단 메타: 다오랩 7기 4조 / 중간점검 / 2026-04-21
- 5명 이름 가로 배치: 할리 · 제이 · 에린 · 누리 · 채드

**이야기하고 싶은 것**
- CAVE는 "지도 없는 곳을 함께 간다"는 은유로 설계된 연구 조직이다
- 오늘 전할 이야기는 "원래 질문을 다시 그린 4주의 기록"이다
- 발표의 프레임(질문 → 재설계 → 조건 지도)을 첫 20초에 깔아둔다

**대사**

## 2. 원래 질문 + 직면한 현실 (50초)

**메시지**: 7주 안에 Yes/No로 답하기엔 구조가 안 맞더라.

**시각 구성 (좌우 분할)**
- 좌: 원래 질문 박스 — **"AI Ops는 자율조직을 강화하는가?"** (Yes/No)
- 우: 현실 박스 3줄
  - 자율조직 = P3~P4 필요
  - CAVE 5명 = **대부분 P1~P2** (누리만 P3)
  - 7주에 P4 도달 = 불가능
- 하단 작은 레전드 바 (재설명 없이 시선만 닿게)
  ```
  P0 미사용 · P1 보조 · P2 협업 · P3 위임 · P4 자율
  ```

**이야기하고 싶은 것**
- 원래 Yes/No 질문으로 진지하게 시작했다 — 도망가거나 축소한 게 아니다
- 자율조직이 관찰되는 임계점(P3~P4)과 CAVE 5명의 실제 구간(P1~P2 중심) 사이에 **구조적 간격**이 있다
- 이 간격을 4주 만에 받아들였고, 그게 다음 슬라이드의 전환으로 이어진다
- Pattern 용어는 다시 설명하지 않되, 하단 레전드 바를 슬쩍 가리켜 청중이 눈으로 복기하게 한다

**대사**
> "시작은 단순한 질문이었습니다. **'AI Ops는 자율조직을 강화하는가.'** Yes냐 No냐로 답하려고 했어요.
> 그런데 4주간 데이터를 모으고 부딪히면서 구조적 한계를 마주했습니다. 자율조직이라고 부를 만한 변화는 보통 **P3 위임 · P4 자율** 단계에서 관찰돼요. (하단 레전드를 가리키며) 그런데 저희 5명의 조직은 **대부분 P1~P2 구간**이고, 누리 조직만 겨우 P3에 닿아 있어요.
> 7주 안에 P4까지 끌어올리는 건 물리적으로 불가능했고, 그렇다고 '아직 잘 모르겠다'로 발표를 끝낼 수도 없었어요. 그래서 **질문을 다시 그렸습니다.**"

**주의**: Pattern 용어는 3/24 발표에서 공유한 전제. 말로는 재설명하지 않되 **하단 레전드 바**를 통해 잊은 분들이 눈으로 복기할 수 있게 한다.

## 3. 질문의 전환 ★ (90초)

**메시지**: Yes/No가 아니라 "무엇이 필요한가(조건)"로 질문을 바꿨다. 이게 이번 중간점검의 메시지다.

**시각 구성**
- 상단 Before/After 대비 박스
  - Before: ~~강화하는가? (Yes/No)~~
  - After: **강화하려면 무엇이 필요한가? (조건)**
- 하단 두 갈래 전략 다이어그램
  - 좌 갈래: **P1~P2 (직접 실험)** — "뭘 하면 자율성이 올라가고, 뭘 하면 감시가 되는가?"
  - 우 갈래: **P3~P4 (인터뷰)** — "이미 도달한 조직은 어떤 조건을 갖췄는가?"
  - 합류점: **「자율조직으로 가는 조건 지도」**
- 최하단 잠정 결론 인용박스
  > "Yes, **투명성 × 주체 방향 × 심리적 안전감**이 갖춰지면 강화한다. 하나라도 빠지면 감시 도구가 된다."

**이야기하고 싶은 것**
- **질문 전환 = 실패의 회피가 아니라 연구의 정교화**라는 프레임을 설득한다
- 두 갈래 전략의 논리: "P1~P2 안에 있는 사람만 할 수 있는 실험" + "이미 도달한 조직에서만 들을 수 있는 이야기"
- 「조건 지도」가 5주 뒤 산출물의 이름이고, 이 발표 전체를 관통하는 목적어다
- 잠정 결론 **세 축**(투명성 · 주체 방향 · 심리적 안전감)을 처음으로 청중에게 박아둔다 — 슬라이드 7·9에서 반복된다

**대사 (3단)**
> **(1) 질문의 재설계**
> "질문을 이렇게 바꿨어요. ~~강화하는가?~~가 아니라 **'강화하려면 무엇이 필요한가?'**. Yes/No 대신 **조건**을 묻는 질문으로 다듬었습니다. 연구를 피한 게 아니라, 7주 안에 답할 수 있는 질문으로 각도를 바꾼 겁니다."
>
> **(2) 두 갈래 전략**
> "그 질문에 답하는 방법은 두 갈래예요. **왼쪽** — 저희 P1~P2 조직 안에서는 **직접 실험**을 돌립니다. '뭘 했을 때 자율성이 올라가는가, 반대로 뭘 하면 감시로 기울어지는가.' **오른쪽** — 이미 P3~P4에 도달한 조직은 **인터뷰**로 접근해요. '어떤 조건을 갖췄기에 거기까지 왔는가.' 같은 현상을 서로 다른 각도에서 관찰하는 거예요."
>
> **(3) 합류점과 잠정 결론**
> "두 길이 만나는 지점에 저희 최종 산출물 **「조건 지도」**가 놓입니다. 잠정 결론은 세 축이에요 — (인용박스를 가리키며) **투명성 · 주체 방향 · 심리적 안전감.** 이 셋이 갖춰지면 AI Ops는 자율을 강화하고, **하나라도 빠지면 감시 도구**로 바뀝니다. 오늘 남은 7분은 이 세 축이 실제 데이터에서 어떻게 드러나는지를 같이 보는 시간입니다."

## 4. 지금 여기 + CAVE 포지션 맵 (90초)

**메시지**: 7기 전체가 P2에 몰려 있고, 감시 체감은 Pattern과 무관하다. CAVE 5명의 분포는 그 축소판이라 교차 비교에 적절하다.

**시각 구성 (상·하 2단)**

*상단 — 7기 기준선 카드 4개*
- 📊 **P2 54%** — P0:0 / P1:3 / **P2:7** / P3:2 / P4:0
- 🔁 **Bottom-up ≥72%** — 개인 36% + 리더·동료 36%
- 📈 **AI 기여 자율성 단조 상승** — P1(3.1) → P2(3.5) → P3(4.5)
- 👁 **감시 체감 Pattern 무관** — 3.3~3.5 일정

*하단 — CAVE 5명 포지션 맵*
```
P0        P1        P2        P3        P4
미사용     보조      협업      위임      자율
          🧑 에린   🧑 할리   🧑 누리
                   🧑 제이
```

**이야기하고 싶은 것**
- Phase 1 기준선 4가지 숫자 중 **감시 체감이 Pattern과 무관**이라는 반직관적 발견이 오늘 발표의 근거 1번
- "단계만 올린다고 감시가 줄어들지 않는다" → 감시를 결정하는 건 **다른 변수** → 그게 조건 지도의 세 축
- CAVE 5명의 분포가 7기 전체와 거의 일치해서 한 조 안에서 **교차 비교 표본**이 성립한다
- 숫자 → 의미(통념 깨기) → 조의 다양성이 왜 강점인지 순서로 읽는다

**대사**
> "Phase 1 기준선부터 간단히 보실게요. 7기 13명 설문에서 나온 **네 가지 숫자**입니다.
> 첫째, **P2 54%** — 대부분 'AI 협업' 단계에 몰려 있어요. 둘째, 도입 경로는 **Bottom-up이 72%** — 개인이나 동료가 먼저 시작하는 패턴이 압도적이에요. 셋째, Pattern이 올라갈수록 **AI 기여 자율성 체감이 단조 상승**합니다 — 3.1에서 3.5, 4.5까지.
> 그런데 **네 번째가 반전**이에요. **감시 체감은 Pattern과 상관이 없어요.** 모든 단계에서 3.3~3.5로 거의 같습니다. 이건 '단계만 올리면 감시가 줄어든다'는 통념이 틀렸다는 뜻이에요. 감시를 결정하는 건 Pattern이 아니라 **다른 변수** — 저희는 그게 조건 지도의 세 축이라고 보고 있습니다.
> (화면 하단을 가리키며) 아래는 CAVE 5명의 현재 포지션이에요. 에린이 P1, 할리와 제이가 P2, 누리가 P3. 이 분포가 7기 전체 분포와 거의 일치해서, **한 조 안에서 교차 비교가 가능한 표본**이 된 게 저희 연구의 출발 자원입니다."

**주의**: 4장 통합으로 정보 밀도 높음. 상단 4카드는 숫자 강조만, 설명은 말로.

## 5. Q2 자율조직 4차원 변화 (80초)

**메시지**: P1 < P2 < P3 단조 상승이 중간 시점에도 재현됐다. 같은 P2 안에서도 "무엇이 변하는가"가 달라진다.

**시각 구성**
- 중앙: 방사형 차트 (축 4개)
  - 의사결정 분산 / 자율적 실행 / 정보 투명성 / 구성원 인식
  - 4개 폴리곤: 에린(P1, 회색) / 할리(P2, 파랑) / 제이(P2, 청록) / 누리(P3, 빨강)
- 우측 요약 박스 3줄
  - ✅ P1(3.0) < P2(3.3) < P3(4.5) 단조 상승
  - ⚠️ 같은 P2인데 할리 vs 제이 패턴이 다름
  - 💡 P3(누리)는 4축 모두 4~5점 포괄 상승

**이야기하고 싶은 것**
- Phase 1에서 본 **단조 상승이 중간 시점에도 재현**되었다 → 관찰이 감에 휘둘리지 않는다
- **같은 Pattern 안에서도 변하는 차원이 다르다** → 자율조직 네 차원은 동시에 움직이지 않음
- 누리 P3의 **포괄 상승**은 Pattern이 성숙하면 네 차원이 함께 올라간다는 신호
- 차트가 처음인 청중을 위해 **읽는 법(축 4개 + 폴리곤 색)**을 한 박자 짚어준다

**대사**
> "이건 중간 시점 Q2 설문을 방사형으로 그린 차트예요. **4축이 각각 자율조직의 네 차원**입니다 — 의사결정 분산, 자율적 실행, 정보 투명성, 구성원 인식. 폴리곤이 크고 바깥쪽으로 갈수록 체감 변화가 큰 거예요.
> 큰 그림부터 말씀드릴게요. **P1 에린(3.0) < P2 할리·제이(3.3) < P3 누리(4.5).** Phase 1에서 봤던 단조 상승이 4주 뒤에도 재현됩니다.
> 더 흥미로운 건 **같은 P2 안의 차이**예요. 할리 폴리곤은 **투명성**만 혼자 튀고, 제이는 **실행과 인식**이 튑니다. 같은 단계여도 어떤 차원이 먼저 움직이는지가 달라요. 저희 보고서 Part 2의 **'같은 P2인데 왜 체감이 다른가'** 질문의 출발점이 바로 이 데이터예요.
> 누리 P3 폴리곤은 4~5점 바깥쪽으로 거의 전면이 넓어져 있죠. **Pattern이 성숙하면 네 차원이 함께 올라간다**는 힌트입니다."

**주의**: 방사형은 PPTX Skill 차트 생성이 불안정할 수 있음 → 이미지로 사전 렌더 후 삽입 예비.

## 6. 🪨 어둠 ↔ 💡 빛 ★ (110초)

2. P3 리더의 어두운 면 — "AI에게만 시킴" (할리)
3. 위임 실패 불안 = 과잉 통제 (에린 4/6 로그)
4. 투명성의 역설 (Q5에서 투명성 0표)

*우측 — 💡 AI Native 5조건 (제이 인터뷰 2건)*
> 인터뷰 대상: **코르카** (AI B2B 스타트업 · 홍영기 제품 총괄) / **Maum AI** (기업용 Physical AI 스타트업 · 서인근 제품 총괄)

1. 비용 제약 제거 (코르카 개인당 월 $300+ / Maum AI 공용 계정 공유)
2. 직무 경계 해체 (전원 기획-개발-배포)
3. 컨텍스트 축적 루프 (diff→정책 자동 업데이트)
4. AX Day — 2주 1회 루틴 (→ Q5 "충분한 학습 시간" 75%의 실체)
5. 서번트 + FOMO 리더십 (리더가 맨 앞에서 달림)

*하단 인용박스*
> "이제야 진짜 애자일 할 수 있는 시대가 왔다." — 홍영기 (코르카)

**이야기하고 싶은 것**
- 왼쪽 **장애물 4개의 공통점** = 기술이 아니라 **사람의 인식과 관계**에서 오는 문제
- 오른쪽 **5조건의 공통점** = 단계별 기술이 아니라 **문화 + 리더십 + 비용 제약 제거**
- **대응쌍**이 핵심 — 인식 부족 ↔ AX Day · 리더 왜곡 ↔ 서번트 리더십 · 과잉 통제 ↔ 직무 경계 해체 · 투명성 역설 ↔ 컨텍스트 축적 루프
- 홍영기 인용은 **기술 낙관이 아니라 "과거 이론(애자일)이 드디어 실행 가능"**이라는 관점을 전한다
- 이 대응쌍이 곧 보고서 Part 3·4의 뼈대이자 조건 지도의 원재료다

**대사 (3단)**
> **(1) 왼쪽 — 🪨 CAVE 4주가 발견한 네 가지 그림자**
> "왼쪽이 저희가 4주간 관찰한 **장애물 네 가지**예요.
> 첫째, 에린 조직의 **'필요성 인식 부족'** — 개인은 AI를 잘 쓰는데 조직 차원 도입은 좀처럼 다음 단계로 넘어가지 않습니다.
> 둘째, 할리가 4팀 비교에서 발견한 **P3 리더의 어두운 면** — 위임한다는 게 사람 대신 AI한테만 시키는 식으로 변질되고 있어요. P3가 감시로 기울어지는 순간입니다.
> 셋째, 에린의 4월 6일 로그에서 드러난 **'위임 실패 불안 = 과잉 통제'** 패턴 — 완벽주의 · 버리지 못함 · ROI 불확실, 세 가지 조합으로 혼자 다 하게 됩니다.
> 넷째, 저희 Q5에서 **투명성을 1위로 뽑은 사람이 한 명도 없었어요.** 이게 역설인데, 투명성이 이미 기본 전제가 된 건지, 아니면 놓치고 있는 건지 — 보고서 Part 3에서 다룰 질문이 여기서 시작됩니다."
>
> **(2) 오른쪽 — 💡 AI Native 조직의 다섯 조건**
> "오른쪽은 제이가 진행한 두 건의 AI Native 조직 인터뷰에서 뽑은 **5조건**이에요. **코르카** — AI B2B 스타트업, 홍영기 제품 총괄 — 과 **마음AI** — 기업용 Physical AI 스타트업, 서인근 제품 총괄. 공통점으로 뽑힌 다섯 가지는 이렇습니다.
> ① **비용 제약 제거** — 코르카는 개인당 월 $300+, 마음AI는 공용 계정으로 심리적 장벽 자체를 없앴어요.
> ② **직무 경계 해체** — 전원이 기획·개발·배포를 같이 돌립니다.
> ③ **컨텍스트 축적 루프** — 매일 새벽 코드 diff가 정책 문서를 자동 업데이트해요.
> ④ **AX Day** — 2주에 한 번, 하던 일 멈추고 2~3명 페어가 6시간 자유 프로젝트를 돕니다.
> ⑤ **서번트 + FOMO 리더십** — 리더가 맨 앞에서 직접 달리면서 구성원의 FOMO를 자극해요."
>
> **(3) 대응쌍 — 왜 이 장이 서사의 한가운데인가**
> "여기서 저희가 주목하는 건 **왼쪽과 오른쪽의 대응 관계**입니다. 저희 Q5 1위가 **'충분한 학습 시간'**이었는데, 그게 **AX Day**라는 구체적 루틴으로 나타나고 있어요. 위임 실패 불안은 **직무 경계 해체**로 풀리고, 리더의 어두운 면은 **서번트 리더십**으로 뒤집힙니다.
> 홍영기 대표의 이 말로 요약돼요 — (인용박스를 가리키며) **'이제야 진짜 애자일 할 수 있는 시대가 왔다.'** 기술 낙관론이 아니에요. 피터 드러커 이후 이론으로만 있던 것이 드디어 실행 가능해졌다는 관점이고, 저희 보고서 Part 3·4는 이 대응쌍을 뼈대로 짤 예정입니다."

## 7. 최종 결과물 — 조건 지도 (50초)

**메시지**: 5주 후에 나올 결과물의 형상은 이렇다.

**시각 구성**
- 상단 제목: **「자율조직으로 가는 조건 지도」**
- 중앙 5파트 목차 (세로 스택)
  - Part 1. 동굴 입구 — 우리의 질문
  - Part 2. 지금 여기 — 우리는 어디에 서 있는가
  - Part 3. 🪨 어둠 속 장애물
  - Part 4. 💡 빛을 찾아서
  - Part 5. 지도 완성 — 조건
- 하단 강조 박스: **투명성 × 주체 방향 × 심리적 안전감**

**이야기하고 싶은 것**
- 5주 뒤 나올 **조건 지도의 5파트 서사 흐름**을 한 줄씩 미리 보여준다
- Part 3·4는 앞 슬라이드 6의 **어둠 ↔ 빛 대응쌍**이 확장된 형태임을 연결시킨다
- 세 축(투명성 · 주체 방향 · 심리적 안전감)을 다시 한 번 못박는다 — 슬라이드 3에서 한 번, 여기서 두 번째, 9에서 세 번째

**대사**
> "5주 뒤 나올 최종 결과물의 모양은 이렇게 **5파트**로 짜입니다.
> **Part 1**은 우리가 왜 질문을 다시 그렸는지를 기록한 동굴 입구, **Part 2**는 Phase 1과 Phase 3 설문 전·후 비교, **Part 3**은 방금 보신 네 개의 그림자를 심화한 장애물 편, **Part 4**는 AI Native 조직 인터뷰에서 뽑은 다섯 빛, 그리고 **Part 5**에서 둘을 겹쳐 만드는 **조건 지도**가 완성됩니다.
> 잠정 결론은 다시 이 세 축이에요 — (박스를 가리키며) **투명성 × 주체 방향 × 심리적 안전감.** 셋 다 갖추면 강화, 하나라도 빠지면 감시. 저희가 남은 3주 동안 증명해야 할 명제입니다."

## 8. 🟡 헬스체크 + 협력 요청 (80초)

**메시지**: 중간 상태는 옐로, 발표 전까지 4가지 도움이 필요하다. (운영진 필수 문항)

**시각 구성 (좌우 분할)**

*좌 — 🟡 헬스체크*
- 🟢 그린: Phase 1 데이터 / 설계 / 서면 인터뷰 회수 / 서사 전환 완료
- 🟡 옐로: 7주 중 4주 경과 — 인터뷰 추가 1~2건 필요 / Phase 3 설문 5/2 일괄 실시 예정
- 🔴 레드: 없음

*우 — 🤝 협력 요청 4*
1. **AI Native 실패 사례 / 백오피스 전환 사례** 추가 인터뷰 소개
2. **P3 단계(AI 위임 운영 중) 조직이 있다면 인터뷰 희망**
3. "AI가 틀렸을 때 어떻게 하나요?" 답할 수 있는 레퍼런스
4. 오늘 발표에 대한 **교차 피드백** — 조건 지도 프레임이 설득력 있는가

**이야기하고 싶은 것**
- 운영진 필수 3문항 중 **헬스체크 + 협력 요청**을 한 슬라이드에 압축 (세 번째 "최종 결과물 형상"은 슬라이드 7에서 이미 답함)
- 헬스체크가 옐로인 **이유를 구체적으로** — 데이터·설계는 안전, 시간과 표본이 타이트
- 협력 요청 네 가지 각각이 **누구에게·무엇을 해주길 바라는지** 명확히
- 청중이 이 자리에서 바로 움직일 수 있도록 — 인터뷰 대상 소개, 레퍼런스 연결 등

**대사**
> "중간점검 필수 답변 드립니다. 헬스체크는 **옐로**예요.
> **🟢 그린** — Phase 1 데이터, 연구 설계, 서면 인터뷰 회수, 그리고 오늘 보여드린 서사 전환까지 안전하게 섰습니다.
> **🟡 옐로** — 7주 중 4주가 지났는데 남은 3주 안에 인터뷰 **1~2건 추가 확보**가 필요하고, Phase 3 설문을 **5월 2일에 각자 조직에서 일괄 실시**해야 해요.
> **🔴 레드** — 현재는 없습니다.
>
> 그래서 오늘 **네 가지 도움 요청**을 드립니다.
> **① AI Native 조직 인터뷰 추가** — 특히 실패 사례나 백오피스 부서가 전환한 케이스가 있으면 소개 부탁드려요.
> **② 혹시 이미 AI 위임이 돌아가는 P3 단계 조직을 아시면** 인터뷰 연결해 주세요. 지금 코르카·마음AI 두 곳은 제품팀이어서, 다른 유형이 필요합니다.
> **③ 'AI가 틀렸을 때 어떻게 대응하나요?'** 이 질문에 답할 수 있는 조직 레퍼런스가 있으면 공유 부탁드려요.
> **④** 마지막으로 오늘 발표 끝나고 **'조건 지도 프레임이 설득력 있는가'** 에 대한 교차 피드백을 받고 싶습니다."
